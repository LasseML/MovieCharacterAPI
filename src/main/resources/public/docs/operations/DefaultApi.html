characterGet

URL

https://localhost:8080/api/v1/character

HTTP Method

GET

Response Type

List

Parameters

characterIdGet

Returns the movie character with the given id

URL

https://localhost:8080/api/v1/character/{id}

HTTP Method

GET

Response Type

ReturnMovieCharacter

Parameters

  • Path: id Long

    ID of movie character to return

characterIdPut

Update the movie character with the given id.

URL

https://localhost:8080/api/v1/character/{id}

HTTP Method

PUT

Response Type

MovieCharacter

Parameters

    movie character object that needs to be added to the store

Parameters

  • Body: body MovieCharacter(MovieCharacter)

    Movie character object that needs to be added to the db
